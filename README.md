# Knight Nest

**The project is dead and no longer working! Don't try to revive it or worse, use it as it is, it has too many vulnerabilities.** (I'm too lazy to fix it)

## Description

Knight chess social network built with TypeScript and Nest.JS.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
